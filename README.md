# 🕹️ My Pygame Project

Проста 2D гра на Python з використанням бібліотеки **pygame**.

---

## 📌 Опис гри

Гравець керує персонажем, уникає ворогів і збирає бонуси.  
Мета — набрати якомога більше очок і не зіткнутися з ворогами.

---

## 🎮 Управління

- ⬆️ Стрілка вверх — рух вгору  
- ⬇️ Стрілка вниз — рух вниз  
- ⬅️ Стрілка вліво — рух вліво  
- ➡️ Стрілка вправо — рух вправо  

---

## 🧩 Ігрові об'єкти

- 👤 Гравець — керований персонаж
- 👾 Вороги — рухаються з правого боку екрана
- ⭐ Бонуси — дають очки при зборі
- 🌄 Фон — анімований рух

---

## 🛠️ Використані технології

- Python 3.x
- pygame
- random
- os
